ℹ️ About This System

This system combines Collaborative Filtering and Sentiment Analysis to provide high-quality product recommendations. First, it uses user-based collaborative filtering to find candidate products similar users liked. Then, it applies NLP sentiment analysis on product reviews to filter and rank recommendations based on positive sentiment ratio.

API Endpoints:

  • POST /api/recommend - Get product recommendations for a user
  • POST /api/sentiment - Analyze sentiment of a review text
  • GET /api/users - Get list of available users
  • GET /api/products - Get list of available products
  • GET /health - System health check
